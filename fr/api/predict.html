Créer une prédiction

Tirez un ou plusieurs futurs échantillonnés d'un profil de données. Nécessite une carte enregistrée.

POST/v1/predict

Corps de la requête

Vous nommez un profil de données et la forme de l'échantillon que vous voulez.

data_profilestringRequis

Le nom de l'un de vos profils, ou d'un profil public tel que temporis/polymarket.

countintegerRequis

Le nombre de trajectoires futures indépendantes à tirer. Doit être au moins 1.

temperaturenumberRequis

Température d'échantillonnage, supérieure à 0. Plus élevée disperse les échantillons ; plus faible les maintient près du chemin le plus probable.

top_pnumberRequis

Seuil d'échantillonnage par noyau, dans la plage de 0 à 1. Plus bas rogne la queue improbable ; 1.0 conserve toute la distribution.

count, temperature et top_p sont expliqués en profondeur dans Prédictions.

Exemple de requête

Ceci tire cinq trajectoires du profil hourly_orders à température neutre :

curl https://api.temporis.co/v1/predict \
  -H "Authorization: Bearer $TEMPORIS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "data_profile": "hourly_orders",
    "count": 5,
    "temperature": 1.0,
    "top_p": 0.9
  }'

Champs de la réponse

Une requête réussie renvoie HTTP 200 avec ce corps.

request_idstring

Un identifiant unique pour cette requête de prédiction.

data_profilestring

Reprend le nom du profil que vous avez demandé.

predict_frominteger

Secondes Unix : le dernier timestamp observé à partir duquel la prévision se poursuit. Chaque trajectoire avance à partir d'ici.

predictionsarray

Les trajectoires échantillonnées. La longueur égale le count que vous avez demandé.

Objet de prédiction

lossnumber

Plus faible signifie que le modèle a jugé cette trajectoire plus typique. Utilisez-la pour classer ou choisir un échantillon représentatif.

dataarray

Les lignes de prévision. Chaque ligne possède un timestamp plus un champ par nom de série dans le profil.

Profils de données publics

Vous pouvez prévoir un jeu de données partagé sans aucune configuration de votre part en passant un nom de profil public comme data_profile. Ceux-ci sont maintenus par Temporis et prêts à servir :

{ "data_profile": "temporis/polymarket", "count": 3, "temperature": 1.0, "top_p": 0.9 }

Erreurs

Les erreurs renvoient le statut HTTP plus un corps JSON de la forme { "detail": "..." }.

Statutmessage detailSignification
400Invalid prediction parameters.Un paramètre est hors plage, comme count inférieur à 1 ou temperature non supérieure à 0.
402Add a credit card before using the API. / There is a payment issue. Add a new credit card.Aucune facturation active. Ajoutez ou corrigez une carte dans le tableau de bord.
404Data profile not found.Aucun profil de ce nom n'existe.
404Data profile is not ready to serve.Le profil existe mais n'a pas terminé sa préparation.
409Not enough data for this data profile.Le profil dispose de moins que son historique minimal requis. Ingérez davantage, puis réessayez.
502Prediction failed.Un échec côté serveur transitoire. Sans danger à réessayer avec un délai progressif.

Consultez Erreurs & codes de statut pour la référence complète.
