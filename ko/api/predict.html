예측 생성

데이터 프로파일에서 하나 이상의 샘플링된 미래를 추출합니다. 등록된 결제 카드가 필요합니다.

POST/v1/predict

요청 본문

데이터 프로파일과 원하는 샘플의 형태를 지정합니다.

data_profilestring필수

여러분의 프로파일 중 하나의 이름, 또는 temporis/polymarket과 같은 공개 프로파일의 이름입니다.

countinteger필수

추출할 독립적인 미래 궤적의 개수입니다. 최소 1 이상이어야 합니다.

temperaturenumber필수

0보다 큰 샘플링 온도입니다. 값이 높을수록 샘플이 넓게 퍼지고, 낮을수록 가장 가능성 높은 경로에 가깝게 유지됩니다.

top_pnumber필수

0에서 1 사이의 뉴클리어스 샘플링 기준값입니다. 값이 낮을수록 가능성이 낮은 꼬리 부분을 잘라내고, 1.0이면 전체 분포를 유지합니다.

count, temperature, top_p에 대한 자세한 설명은 예측에 있습니다.

응답 필드

요청이 성공하면 다음 본문과 함께 HTTP 200을 반환합니다.

request_idstring

이 예측 요청의 고유 식별자입니다.

data_profilestring

요청한 프로파일 이름을 그대로 반환합니다.

predict_frominteger

Unix 초 단위: 예측이 이어지는 마지막 관측 타임스탬프입니다. 모든 궤적은 이 지점에서 앞으로 전개됩니다.

predictionsarray

샘플링된 궤적입니다. 길이는 요청한 count와 같습니다.

Prediction 객체

lossnumber

값이 낮을수록 모델이 이 궤적을 더 전형적이라고 판단했음을 의미합니다. 대표 샘플을 순위 매기거나 고를 때 사용하세요.

dataarray

예측 행입니다. 각 행에는 timestamp와 프로파일의 시계열 이름마다 하나의 필드가 있습니다.

공개 데이터 프로파일

공개 프로파일 이름을 data_profile로 전달하면 별도의 설정 없이 공유 데이터셋으로 예측할 수 있습니다. 이러한 프로파일은 Temporis가 관리하며 바로 사용할 수 있습니다:

{ "data_profile": "temporis/polymarket", "count": 3, "temperature": 1.0, "top_p": 0.9 }

오류

오류는 HTTP 상태와 함께 { "detail": "..." } 형식의 JSON 본문을 반환합니다.

상태detail 메시지의미
400Invalid prediction parameters.count가 1 미만이거나 temperature가 0보다 크지 않은 경우 등, 파라미터가 허용 범위를 벗어났습니다.
402Add a credit card before using the API. / There is a payment issue. Add a new credit card.활성 결제 수단이 없습니다. 대시보드에서 카드를 추가하거나 수정하세요.
404Data profile not found.해당 이름의 프로파일이 존재하지 않습니다.
404Data profile is not ready to serve.프로파일은 존재하지만 아직 준비가 완료되지 않았습니다.
409Not enough data for this data profile.프로파일의 이력이 최소 요구치보다 적습니다. 더 수집한 후 다시 시도하세요.
502Prediction failed.일시적인 서버 측 오류입니다. 백오프를 적용해 재시도해도 안전합니다.

전체 레퍼런스는 오류 및 상태 코드를 참고하세요.
