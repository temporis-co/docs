Temporis 문서

예측, API로.

Temporis는 여러분의 시계열을 확률적 예측으로 바꿔 줍니다. 숫자를 스트리밍으로 보내고, 그 데이터를 어떻게 읽어야 하는지 데이터 프로파일로 설명한 다음, 하나의 엔드포인트만 호출하면 다음에 무엇이 올지 예측이 생성됩니다 — 학습할 모델도, 운영할 인프라도 필요 없습니다.

Temporis 워크플로의 형태

Temporis의 모든 것은 세 가지 명사를 중심으로 돌아갑니다. 이 개념들이 머릿속에서 자리 잡으면 전체 API가 자연스럽게 따라옵니다.

데이터 소스
원시 관측값을 밀어 넣는 이름이 붙은 스트림으로, (timestamp, name, value) 행으로 이루어진 테이블과 비슷합니다. 하나의 데이터 소스는 여러 시계열을 담을 수 있습니다.
데이터 프로파일
Temporis에게 데이터 소스를 어떻게 읽어야 하는지 알려 주는 저장된 레시피입니다: 어떤 시계열을, 어떤 간격으로, 얼마나 많은 이력으로, 그리고 어떻게 정제할지를 정의합니다. 예측은 바로 이 프로파일을 대상으로 수행됩니다.
예측
데이터 프로파일을 지정해 /v1/predict를 한 번 호출하면 여러 개의 가능한 미래가 반환되며, 각각은 앞으로 이어지는 값들의 완전한 궤적입니다.

데이터 소스와 데이터 프로파일은 Temporis 대시보드에서 관리하고, 대량 작업 — 데이터 전송과 예측 요청 — 은 여기에 문서화된 API를 통해 수행합니다.

맛보기

관측값을 보내고 예측을 요청하는 모습은 다음과 같습니다:

# 1. Send observations to a data source
curl -X POST "https://api.temporis.co/v1/data_sources/ingest" \
  -H "Authorization: Bearer $TEMPORIS_TOKEN" \
  --json '{
    "data_source": "store_metrics",
    "records": [
      { "timestamp": 1716249600, "name": "orders", "value": 42 },
      { "timestamp": 1716253200, "name": "orders", "value": 51 }
    ]
  }'

# 2. Ask a data profile for the next steps
curl -X POST "https://api.temporis.co/v1/predict" \
  -H "Authorization: Bearer $TEMPORIS_TOKEN" \
  --json '{
    "data_profile": "hourly_orders",
    "count": 5,
    "temperature": 1.0,
    "top_p": 0.9
  }'
import os, requests

BASE = "https://api.temporis.co"
headers = {"Authorization": f"Bearer {os.environ['TEMPORIS_TOKEN']}"}

# 1. Send observations to a data source
requests.post(f"{BASE}/v1/data_sources/ingest", headers=headers, json={
    "data_source": "store_metrics",
    "records": [
        {"timestamp": 1716249600, "name": "orders", "value": 42},
        {"timestamp": 1716253200, "name": "orders", "value": 51},
    ],
}).raise_for_status()

# 2. Ask a data profile for the next steps
forecast = requests.post(f"{BASE}/v1/predict", headers=headers, json={
    "data_profile": "hourly_orders",
    "count": 5,
    "temperature": 1.0,
    "top_p": 0.9,
}).json()

print(forecast["predictions"][0]["data"])
const BASE = "https://api.temporis.co";
const headers = {
  "Authorization": `Bearer ${process.env.TEMPORIS_TOKEN}`,
  "Content-Type": "application/json",
};

// 1. Send observations to a data source
await fetch(`${BASE}/v1/data_sources/ingest`, {
  method: "POST",
  headers,
  body: JSON.stringify({
    data_source: "store_metrics",
    records: [
      { timestamp: 1716249600, name: "orders", value: 42 },
      { timestamp: 1716253200, name: "orders", value: 51 },
    ],
  }),
});

// 2. Ask a data profile for the next steps
const res = await fetch(`${BASE}/v1/predict`, {
  method: "POST",
  headers,
  body: JSON.stringify({
    data_profile: "hourly_orders",
    count: 5,
    temperature: 1.0,
    top_p: 0.9,
  }),
});
const forecast = await res.json();
console.log(forecast.predictions[0].data);
