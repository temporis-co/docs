Criar uma previsão

Extraia um ou mais futuros amostrados de um perfil de dados. Requer um cartão cadastrado.

POST/v1/predict

Corpo da requisição

Você nomeia um perfil de dados e o formato da amostra que deseja.

data_profilestringObrigatório

O nome de um dos seus perfis, ou um perfil público como temporis/polymarket.

countintegerObrigatório

O número de trajetórias futuras independentes a extrair. Deve ser pelo menos 1.

temperaturenumberObrigatório

Temperatura de amostragem, maior que 0. Valores mais altos espalham as amostras; valores mais baixos as mantêm próximas do caminho mais provável.

top_pnumberObrigatório

Limite de amostragem por núcleo (nucleus sampling), no intervalo de 0 a 1. Valores mais baixos cortam a cauda improvável; 1.0 mantém a distribuição completa.

count, temperature e top_p são explicados em profundidade em Previsões.

Campos da resposta

Uma requisição bem-sucedida retorna HTTP 200 com este corpo.

request_idstring

Um identificador único para esta requisição de previsão.

data_profilestring

Repete o nome do perfil que você solicitou.

predict_frominteger

Segundos Unix: o último timestamp observado a partir do qual a previsão continua. Cada trajetória avança a partir daqui.

predictionsarray

As trajetórias amostradas. O comprimento é igual ao count que você solicitou.

Objeto de previsão

lossnumber

Quanto menor, mais típica o modelo considerou esta trajetória. Use-o para classificar ou escolher uma amostra representativa.

dataarray

As linhas da previsão. Cada linha tem um timestamp mais um campo por nome de série no perfil.

Perfis de dados públicos

Você pode fazer previsões sobre um conjunto de dados compartilhado sem nenhuma configuração própria, passando o nome de um perfil público como data_profile. Estes são mantidos pela Temporis e estão prontos para servir:

{ "data_profile": "temporis/polymarket", "count": 3, "temperature": 1.0, "top_p": 0.9 }

Erros

Os erros retornam o status HTTP mais um corpo JSON no formato { "detail": "..." }.

Statusmensagem detailSignificado
400Invalid prediction parameters.Um parâmetro está fora do intervalo, como count abaixo de 1 ou temperature não maior que 0.
402Add a credit card before using the API. / There is a payment issue. Add a new credit card.Sem cobrança ativa. Adicione ou corrija um cartão no painel.
404Data profile not found.Não existe nenhum perfil com esse nome.
404Data profile is not ready to serve.O perfil existe, mas ainda não terminou de ser preparado.
409Not enough data for this data profile.O perfil tem menos do que o histórico mínimo exigido. Ingira mais e tente novamente.
502Prediction failed.Uma falha transitória no servidor. Seguro tentar novamente com backoff.

Consulte Erros e códigos de status para a referência completa.
